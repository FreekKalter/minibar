WanneerKanIkWinnen.nl

Welkom op wanneerkanikwinnen.nl! Je kent die acties wel waarbij een bedrijf elk uur een prijs verloot. Je moet dan vaak een code invoeren die je bij dat product krijgt. Grolsch heeft zo'n actie (vaak tijdens het wk/ek) waarbij ze elk uur een minibar weggeven. Als je zo'n actie code hebt wil je deze natuurlijk invoeren op het moment dat je de meeste kans maakt. In andere woorden, wanneer er zo min mogelijk mensen hun code invoeren. Op deze site kun je precies zien wanneer jij de meeste kans maakt!

Op onderstaande heatmap zie je in 1 oogopslag welke tijden populair zijn, deze vakjes zijn donker groen gekleurd. De uren dat er weinig mensen hun code invoeren zijn geel.
De grafiek daaronder geeft een gedetaileerde weergave van een dag. Klik in de heatmap op een dag om de gedetaileerde grafiek van die dag te laten zien.
