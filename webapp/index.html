WanneerKanIkWinnen.nl

Welkom op wanneerkanikwinnen.nl! Je kent die acties wel waarbij een bedrijf elk uur een prijs verloot onder klanten die dat uur een actie code invoeren. Grolsch heeft zo'n actie (vaak tijdens het wk/ek) waarbij ze elk uur een minibar weggeven. Als je zo'n actie code hebt wil je deze natuurlijk invoeren op het moment dat je de meeste kans maakt. Oftewel wanneer er zo min mogelijk mensen hun code invoeren. Op deze site kun je precies zien wanneer jij de meeste kans maakt!

Op onderstaande heatmap zie je in 1 oogopslag welke tijden populair zijn, deze vakjes zijn donker groen gekleurd. De uren dat er weinig mensen hun code invoeren zijn geel.

Onderstaande grafiek geeft een gedaileerde weergave van een dag. Klik in de bovenstaande heatmap op een dag om de gedaileerde grafiek van die dag te laten zien.
